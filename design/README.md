# Design Sources

Сюда складываем исходные материалы дизайна:

- PDF-макеты
- изображения
- шрифты
- референсы

Рекомендуемое имя для текущего макета:

- `design/wedding-invitation-mockup.pdf`

Сейчас в проекте уже есть:

- `design/wedding-invitation-mockup.pdf`
- `design/extracted/` - автоматически извлечённые ассеты из PDF

`design/extracted/` нужен как вспомогательный референс для сборки сайта, но главным источником решений по-прежнему остаётся сам PDF.
