# Wedding Invitation Website

Проект электронного свадебного приглашения.

На текущем этапе мы уже собрали базовый контекст, загрузили PDF-макет и зафиксировали стартовые правила, чтобы дальше двигаться последовательно и без потери решений.

## Структура

- `docs/PROJECT_CONTEXT.md` - актуальный контекст проекта
- `docs/PROJECT_RULES.md` - продуктовые и технические правила
- `docs/ROADMAP.md` - рабочий план и этапы
- `docs/OPEN_QUESTIONS.md` - вопросы и недостающие данные
- `docs/CONTENT_STRUCTURE.md` - карта блоков сайта по PDF-макету
- `docs/DESIGN_NOTES.md` - визуальное направление и заметки по стилю
- `design/README.md` - куда складывать макеты и исходники

## Ближайший фокус

1. Закрыть открытые вопросы по текстам, контактам и точному таймингу.
2. Выбрать стек и зафиксировать стартовую архитектуру.
3. Начать сборку сайта по уже описанной структуре блоков.
